
  # Devorah Aberman Portfolio

This repository contains the source code for my personal portfolio website. The site is built from a Figma design and showcases my work, case studies, and design process.

## About

This portfolio is a modern web presentation of my design projects and professional experience. It was created to highlight:

- selected work in UX/UI and accessibility
- project summaries and case studies
- professional design skills and tools
- responsive, polished presentation for both desktop and mobile

## Built With

- Vite
- React
- TypeScript
- Tailwind CSS
- Motion for animation
- Radix UI components

## Design Origins

The layout and visual styling were adapted from a Figma design: https://www.figma.com/design/V8TsHLUX2oebDcgImivRa2/Designer-Portfolio-Page

## Project Structure

- `src/app/components/` — React components used across the site
- `src/app/` — application entry and page layout
- `src/imports/` — static image assets used by project cards
- `src/styles/` — custom global and theme styles
- `vite.config.ts` — Vite configuration and asset handling

## Local Development

To run the portfolio locally:

```bash
pnpm install
pnpm dev
```

Then open the local development URL shown in the terminal.

## Build

To create a production build:

```bash
pnpm build
```

## Notes

This repository is intended to showcase my portfolio and the front-end implementation of the site. It includes custom asset handling and a responsive card-based layout for presenting case studies.
